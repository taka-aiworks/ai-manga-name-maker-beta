- **ターゲット**: 初心者向け（漫画制作未経験者）

```
1. ネーム制作ツール（現在のツール）
   ↓ キャラ設定・ポーズ・表情データ
2. プロンプト生成ツール（連携予定）

3. AI画像生成（ChatGPT/Midjourney等）
   ↓ 完成画像
4. 画像合成・最終調整
   ↓
5. 完成漫画
```
